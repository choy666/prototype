# Documentación del Proyecto

## 📚 Guías Disponibles

### [01-guia-rapida.md](./01-guia-rapida.md)

Overview del proyecto, configuración inicial y flujo general de la aplicación.

### [02-integracion-tiendanube.md](./02-integracion-tiendanube.md)

Guía de testing suite, deploy en producción, monitoreo y mantenimiento del sistema.

## 🚀 Quick Start

1. Leer [01-guia-rapida.md](./01-guia-rapida.md) para configuración inicial
2. Configurar integraciones según las guías específicas:
   - Tiendanube: [02-integracion-tiendanube.md](./02-integracion-tiendanube.md)
   - Mercado Libre: [03-integracion-mercadolibre.md](./03-integracion-mercadolibre.md)
3. Usar el panel admin con [04-guia-administracion.md](./04-guia-administracion.md)
4. Deploy a producción con [05-testing-produccion.md](./05-testing-produccion.md)
