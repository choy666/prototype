# COMANDOS EXACTOS PARA CORREGIR WEBHOOK SECRET
# Copia y pega estos comandos en orden:

# 1. Ir al dashboard de MercadoPago y obtener el webhook secret real:
# https://www.mercadopago.com.ar/developers/panel/applications
# → Tu aplicación → Webhooks → Copiar el secret de la URL del webhook

# 2. Agregar el secret correcto en Vercel (producción):
vercel env add MERCADO_PAGO_WEBHOOK_SECRET production
# (Pegar el secret real cuando te lo pida)

# 3. Verificar que esté configurado:
vercel env ls production | grep MERCADO_PAGO_WEBHOOK_SECRET

# 4. Sincronizar localmente:
vercel env pull .env.local

# 5. REDESPLIEGUE CRÍTICO (obligatorio):
vercel --prod

# 6. Verificar actualización:
node scripts/debug-webhook-secret.js

# 7. Probar con webhook real desde el dashboard de MercadoPago
